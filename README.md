# JavaLab3
```Завдання:```
Створити клас, який складається з виконавчого методу, що виконує дію над 
текстовим рядком (В усіх питальних реченнях заданого тексту знайти та надрукувати без
повторень слова заданої довжини.), тип якого визначено варіантом (StringBuffer). Необхідно
обробити всі виключні ситуації, що можуть виникнути під час виконання
програмного коду. Всі змінні повинні бути описані та значення їх задані у
виконавчому методі.
